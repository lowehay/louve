# Louve - E-commerce Dashboard

A modern, responsive e-commerce dashboard built with Bootstrap 5, featuring a collapsible sidebar, interactive charts, and a clean UI.

## Features

- Responsive design
- Collapsible sidebar
- Interactive charts using ApexCharts
- Clean and modern UI
- Dark/light mode ready

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- [Bootstrap 5](https://getbootstrap.com/)
- [ApexCharts](https://apexcharts.com/)
- [Boxicons](https://boxicons.com/)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/louve-dashboard.git
   cd louve-dashboard
   ```

2. Open `html/index.html` in your browser.
